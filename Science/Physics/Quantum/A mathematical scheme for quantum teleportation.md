Отлично! Пройдём через **математическую схему квантовой телепортации** шаг за шагом. Предположим, что ты знаком с основами квантовой механики: суперпозициями, тензорным произведением состояний, базисом Белла и уравнениями для квантовых состояний. Если что-то будет неясно — уточняй, разберём подробнее.

---
## **Краткое напоминание цели**
Алиса хочет **передать квантовое состояние**:
$$\psi\rangle_M = \alpha |0\rangle + \beta |1\rangle$$
от некоторой частицы MM (message) Бобу, у которого есть только одна частица BB, **запутанная** с частицей AA, находящейся у Алисы.  
Алиса и Боб используют:
- Запутанную пару: $\Phi^+\rangle_{AB} = \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)$,
- Измерение в **базисе Белла** над $M$ и $A$,    
- Классическую передачу 2 бит информации (результат измерения),    
- Квантовое преобразование у Боба.    
---
## **1. Начальное состояние всей системы**
Обозначим:
- $M$ — частица со **состоянием для телепортации**,    
- $A$ — часть запутанной пары (у Алисы),    
- $B$ — часть запутанной пары (у Боба).    

Полное начальное состояние трёх частиц:
$$\psi\rangle_M \otimes |\Phi^+\rangle_{AB} = \left( \alpha|0\rangle + \beta|1\rangle \right)_M \otimes \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)_{AB}$$

Распишем это:
$$\Psi\rangle_{MAB} = \frac{1}{\sqrt{2}} \left[ \alpha|0\rangle_M(|00\rangle + |11\rangle)_{AB} + \beta|1\rangle_M(|00\rangle + |11\rangle)_{AB} \right]$$

Раскроем скобки:
$$\Psi\rangle_{MAB} = \frac{1}{\sqrt{2}} \left[ \alpha|0\rangle|0\rangle|0\rangle + \alpha|0\rangle|1\rangle|1\rangle + \beta|1\rangle|0\rangle|0\rangle + \beta|1\rangle|1\rangle|1\rangle \right]$$
(Порядок: $M,A,B$)

---
## **2. Представим состояние в базе Белла**
Теперь наша цель — **представить первые две кубитные компоненты** $M$ и $A$ в **базисе Белла**, а не в стандартном.

Базис Белла:
$$|\Phi^\pm\rangle = \frac{1}{\sqrt{2}}(|00\rangle \pm |11\rangle), \quad |\Psi^\pm\rangle = \frac{1}{\sqrt{2}}(|01\rangle \pm |10\rangle)$$
Здесь работает **приём замены базиса**: нужно переписать тензорное произведение $\psi\rangle_M \otimes |\Phi^+\rangle_{AB}$ в виде линейной комбинации состояний Белла на $MA$ и соответствующих состояний частицы $B$.

После громоздких, но прямолинейных алгебраических преобразований (если хочешь, можем пройти их вручную), получаем:
$$|\Psi\rangle_{MAB} = \frac{1}{2} \Big[ |\Phi^+\rangle_{MA} \otimes (\alpha|0\rangle + \beta|1\rangle)_B + |\Phi^-\rangle_{MA} \otimes (\alpha|0\rangle - \beta|1\rangle)_B + ∣Ψ+⟩MA⊗(β∣0⟩+α∣1⟩)B+∣Ψ−⟩MA⊗(β∣0⟩−α∣1⟩)B]+ |\Psi^+\rangle_{MA} \otimes (\beta|0\rangle + \alpha|1\rangle)_B + |\Psi^-\rangle_{MA} \otimes (\beta|0\rangle - \alpha|1\rangle)_B \Big]$$
То есть: **вся система — суперпозиция четырёх состояний Белла** на частицах $M, A$, и соответствующих **изменённых состояний** у частицы Боба $B$.

---
## **3. Алиса делает измерение в базе Белла**
Теперь Алиса измеряет частицы MM и AA в **базисе Белла**.

Результат измерения — один из четырёх исходов:
- $|\Phi^+\rangle$
- $|\Phi^-\rangle$
- $|\Psi^+\rangle$
- $|\Psi^-\rangle$

После измерения:
- Вся система **схлопывается** в один из четырёх слагаемых выше,    
- Частица Боба оказывается в **одном из четырёх состояний**, **связанном с исходным ∣ψ⟩|\psi\rangle**, но, возможно, подвергнутом унитарному преобразованию.
---
## **4. Боб применяет квантовую операцию**
Когда Боб узнаёт, **какой результат измерения** получила Алиса (по классическому каналу), он применяет **одну из четырёх унитарных операций**:

| Результат Алисы   | Состояние Боба | Что делать? |
| ----------------- | -------------- | ----------- |
| $(\Phi^+\rangle)$ | $(\alpha)$     |             |
| $(\Phi^-\rangle)$ | $(\alpha)$     |             |
| $(\Psi^+\rangle)$ | $(\beta)$      |             |
| $(\Psi^-\rangle)$ | $(\beta)$      |             |

Где:
- **X** — оператор Паули-X (квантовый аналог NOT),    
- **Z** — оператор Паули-Z (меняет знак у $|1\rangle$).

После правильной коррекции Боб **восстанавливает исходное состояние $\psi\rangle = \alpha|0\rangle + \beta|1\rangle$!

---
## **5. Ключевые особенности**
- **Оригинальная частица не передаётся** — только состояние.    
- **Нельзя использовать это для передачи информации быстрее света**, потому что Боб не может ничего узнать о состоянии, пока не получит классические биты от Алисы.    
- Это **не клонирование** — оригинальное состояние **уничтожается при измерении** у Алисы.
