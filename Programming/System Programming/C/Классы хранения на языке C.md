Классы хранения в C определяют, как переменные хранятся, используются и поддерживаются в памяти, каждый из которых влияет на область видимости, время жизни и связывание переменной.
##### `auto` → Локальная область видимости, используется по умолчанию.
* Локальные переменные внутри функций без явного указания класса хранения являются auto.
* Они существуют только во время выполнения функции.
##### `register` → Переменная хранится в регистре процессора.
* Используется для оптимизации доступа к переменной.
* Компилятор может проигнорировать этот запрос, если регистры заняты.
##### `static` → Сохраняет значение переменной между вызовами функции, имеет внутреннее связывание.
* Если объявлена внутри функции, её значение сохраняется между вызовами.
* Если объявлена вне функции, доступна только в этом файле.
##### `extern` → Используется для глобальных переменных и функций с внешним связыванием.
* Позволяет ссылаться на переменную, определённую в другом файле.
* Объявление extern int x; указывает, что x определена в другом месте.
