`sizeof` возвращает **размер в байтах типа или объекта**, причем вычисляется на этапе компиляции.
`strlen` вычисляет длину строки **в рантайме**, возвращает **количество символов до `'\0'`**, а не байтов (но это совпадает для `char`).

Если `int` больше, чем `char` может вместить, сохраняются **младшие байты** (обычно `char` — это 8 бит).
```c
int x = 300;  
char c = (char)x;  
printf("%d\n", c);  // Вероятно, 44 (так как 300 % 256 = 44)
```

- `const int *p` → **указатель на `const` int** (значение менять нельзя, сам указатель — можно).
- `int *const p` → **константный указатель на `int`** (сам указатель менять нельзя, значение — можно).
- `const int *const p` → **константный указатель на `const` int** (нельзя менять ни указатель, ни значение).

- `malloc(n)` – выделяет `n` байт, но не инициализирует память.
- `calloc(n, size)` – выделяет `n * size` байт и **инициализирует нулями**.
- `realloc(ptr, new_size)` – изменяет размер ранее выделенной памяти, возможно с перемещением.

Undefined Behavior (UB) - Неопределенное поведение
```c
int x;
printf("%d\n", x);  // Использование неинициализированной переменной → UB
```

порядок вызова функций может быть неопределенным, это зависит от компиляции
Strict Aliasing Rule - В C указатели на **разные типы (не `char *`)** не должны ссылаться на один и тот же объект.
**Пример нарушения strict aliasing:**
```C
float f = 3.14;
int *p = (int*)&f;  // UB! Нарушение strict aliasing
```

`volatile` запрещает компилятору **оптимизировать чтение/запись переменной**.
**Когда нужно?**
- Работа с аппаратурой (`volatile int *port = (int*)0x4000;`)
- Глобальные переменные в `signal` и `interrupt` обработчиках.
```c
volatile int flag = 0;
while (!flag) { }  // Без volatile компилятор может удалить этот цикл
```

 Массив — структура данных фиксированного размера. Его элементы должны быть одного типа. Например: [10, 12, 85, 27]. Массивы статичны: нельзя изменить размер или тип элементов после объявления.

Список — динамическая структура данных. В нем можно хранить элементы разных типов. Такая структура есть в Python. Пример списка: [0,”red”, 8.5, False].
