##### Работа с файлами в C позволяет создавать, открывать, читать, записывать и закрывать файлы с помощью таких функций. 
`fopen()` — открытие файла в нужном режиме
`fwrite() / fprintf()` — запись данных
`fread() / fscanf()` — чтение данных
`fclose()` — закрытие файла
