### **Базовые**
**`\n` Новая строка (newline). Перемещает курсор на начало следующей строки**
**`\t` Горизонтальная табуляция (tab). Вставляет символ табуляции**
**`\v`  Вертикальная табуляция. Перемещает курсор на следующую вертикальную позицию**
**`\0` Нулевой символ (null character). Используется для завершения строк**
**`\a`  Звуковой сигнал (alert). Вызывает звуковой сигнал на устройстве вывода**
**`\b`  Backspace. Перемещает курсор на одну позицию назад**
**`\r`  Возврат каретки (carriage return). Перемещает курсор в начало строки**
**`\\` Обратный слэш. Вставляет символ \**
`\"`  Двойная кавычка. Вставляет символ "**
**`\'`Одинарная кавычка. Вставляет символ '**
**`\?` Вопросительный знак. Используется для вставки ? в триграфах**
### **Ansi**
**`\033[?25l` Выключить курсор**
**`\033[?25h` Включить курсор**
**`\033[2J`Очистить экран**
**`\033[<y>;<x>H` Переместить курсор**
