Одним из наиболее важных понятий в языке программирования является возможность выполнения кода в зависимости от выполнения определенных условий. Современный Фортран поддерживает это с помощью **if**- и **select**-выражений, которые описаны в этом разделе.

Простейшая форма if-заявок в Фортране имеет следующий синтаксис
```fortran
if (scalar-logical-expr) then
    block
end if
```
где **scalar-logical-expr** - это булево выражение, которое должно быть оценено как истинное (**.true.**), чтобы операторы в **block** были выполнены. Расширенная версия if-выражения добавляет **else**-блок со следующим синтаксисом
```fortran
if (scalar-logical-expr) then
    block1
else
    block2
end if
```

В этой форме **блок1** будет выполнен, если **scalar-logical-expr** будет оценен как true, иначе будет выполнен **блок2**. Третья форма if-выражения содержит один или несколько **else if**-выражений со следующим синтаксисом:
```fortran
if (scalar-logical-expr1) then
    block1
else if (scalar-logical-expr2) then
    block2
else
    block3
end if
```

В этой форме сначала оценивается **scalar-logical-expr1**. Если это выражение равно true, то выполняется **блок1**, иначе, если **scalar-logical-expr2** оценивается как true, то выполняется **блок2**. Если никакие другие выражения не имеют значения true, то выполняется **блок3**. Условие if может содержать несколько **else if**-блоков. Использование if-выражений показано в следующем примере:
```fortran
program logic
     implicit none
     
     integer :: x
     logical :: flag
     ! Считывание целого числа со стандартного ввода
	 
    write(*,*) 'Enter an integer value.'
    read(*,*) x
	
    ! Корректное значение в интервале 0-1000
	
    flag = .FALSE.
	
    if (x>1000) then
        x = 1000
        flag = .TRUE.
    end if
	
    if (x<0) then
        x = 0
        flag = .TRUE.
    end if
	
	! Если флаг равен .TRUE, то входное значение
    ! было исправлено.
	
    if (flag) then
        write(*,'(a,I4)') 'Corrected value = ', x
    else
        write(*,'(a,I4)') 'Value = ', x
    end if
	
    stop
end program logic
```

Еще одной условной конструкцией является утверждение case.
```fortran
select case (expression)
    case selector
        block
end select
```

В этом операторе выражение **выражение** оценивается и выполняется **case**-блок с соответствующим **селектором**. Для обработки случая, когда ни один **case**-блок не соответствует **expr**, может быть добавлен **case**-блок с ключевым словом **default**. Тогда синтаксис становится таким:
```fortran
select case (expr)
    case selector
        block
    case default
        block
end select
```

Пример использования падежных окончаний приведен в следующем примере:
```fortran
select case (display_mode)
    case (displacements)
        ...
    case (geometry)
        ...
end select
```
Для обработки случая, когда **display_mode** не соответствует ни одной из альтернатив, приведенный выше код модифицируется в следующий.
```fortran
select case (display_mode) case (displacements)
        ...
    case (geometry)
        ...
    case default
        ...
end select
```
Следующий пример программы иллюстрирует использование прецедентных высказываний.
```fortran
program case_sample
    integer :: value
	 
    write(*,*) 'Enter a value'
    read(*,*) value
	 
    select case (value)
        case (:0)
            write(*,*) 'Less than one.'
        case (1)
            write(*,*) 'Number one!'
        case (2:9)
            write(*,*) 'Between 2 and 9.'
        case (10)
            write(*,*) 'Number 10!'
        case (11:41)
            write(*,*) 'Less than 42 but greater than 10.'
        case (42)
            write(*,*) 'Meaning of life or perhaps 6*7.'
        case (43:)
            write(*,*) 'Greater than 42.'
        case default
            write(*,*) 'This should never happen!'
    end select
	
    stop
end program case_sample
```
