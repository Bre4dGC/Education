Самым распространенным повторяющимся оператором в Фортране является **do**-выражение. Его синтаксис следующий:
```fortran
do variable = expr1, expr2, expr3
    block
end do
```
**variable** - управляющая переменная цикла. **expr1** - начальное значение, **expr2** - конечное значение и **expr3** - интервал шага. Если интервал шага не задан, он принимается равным 1. Существует два способа управления потоком выполнения в **do**-выражении. Команда **exit** завершает цикл, и выполнение программы продолжается после оператора **do**. Команда **cycle** завершает выполнение текущего блока и продолжает выполнение со следующим значением управляющей переменной. Пример ниже иллюстрирует использование команды **do**.

```fortran
program loop_sample
	
    implicit none
	
    integer :: i
	
    do i=1,20
        if (i>10) then
            write(*,*) 'Terminates do-statement.'
            exit
        else if (i<5) then
            write(*,*) 'Cycling to next value.'
            cycle
        end if
        write(*,*) i
    end do
	
    stop
	
end program loop_sample
```

Приведенная выше программа дает следующий результат:
```fortran
Cycling to next value.
Cycling to next value.
Cycling to next value.
Cycling to next value.
5
6
7
8
9
10
Terminates do-statement.
```
Еще одним доступным повторяющимся оператором является оператор **do while**. С помощью этого оператора блок кода может выполняться до тех пор, пока не будет выполнено определенное условие. Синтаксис следующий:
```fortran
do while (scalar-logical-expr)
    block
end do
```
В следующем коде показан простой **do while**-статью, печатающую функцию $f(x) = \sin(x)$
```fortran
x = 0.0
do while x<1.05
    f = sin(x)
    x = x + 0.1
    write(*,*) x, f
end do
```
Другие повторяющиеся операторы, такие как **forall** и **where**, рассматриваются в разделах, посвященных возможностям массивов.
