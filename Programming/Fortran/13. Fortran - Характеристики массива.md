### Утверждения "За" и "Где"
В Fortran добавлено несколько новых выражений цикла. Для оптимизации вложенных циклов для выполнения на многопроцессорных машинах был добавлен оператор **forall**. Синтаксис следующий:
```fortran
forall (index = lower:upper [,index = lower:upper])
    [body]
end forall
```

В следующем примере показано, как можно заменить **do**-выражение на **forall**-выражение.
```fortran
do i = 1, n
    do j = 1, m
        A(i, j) = i + j
    end do
end do
```

```fortran
! Эквивалентно
forall(i = 1:n, j = 1:m)
    A(i, j) = i + j
end forall
```

Еще одним оператором, оптимизированным для многопроцессорных архитектур, является **where**-выражение. С помощью этого оператора можно эффективно выполнять условные операции над массивом. Синтаксис существует в двух вариантах.
```fortran
where (logical-array-expr)
    array-assignments
end where

and

where (logical-array-expr)
    array-assignments
else where
    array-assignments
end where
```

Использование **where**-выражения лучше всего проиллюстрировать на примере.
```fortran
where (A > 1)
    B = 0
else where
    B = A
end where
```
В этом примере в выражении **where** используются два массива одинакового размера. В этом случае значениям в массиве **B** присваивается 0, если элемент в массиве A больше 1, в противном случае элементу в **B** присваивается то же значение, что и в массиве **A**.
