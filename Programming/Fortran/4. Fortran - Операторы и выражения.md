В Фортране определены следующие арифметические операторы:

| Оператор | Описание   |
| -------- | ---------- |
| **       | Экспонента |
| *        | Умножение  |
| /        | Деление    |
| +        | Сложение   |
| -        | Вычитание  |
Скобки используются для указания порядка выполнения различных операторов. Если в выражении не указаны скобки, операторы оцениваются в следующем порядке:
1. Операции с **
2. Операции с * или /
3. Операции с + или -

Следующий код иллюстрирует приоритет операторов.
```fortran
c = a + b / 2   ! Эквивалентно
c = (a + b) / 2 ! В этом случае вычисляется
```
Эквивалентно $a + (b / 2)$
В этом случае вычисляется $(a + b)$, а затем $/ 2$

Реляционные операторы:

| Operator   | Description      |
| ---------- | ---------------- |
| < or .lt.  | меньше           |
| <= or .le. | меньше или равно |
| > or .gt.  | больше           |
| >= or .ge. | больше или равно |
| == or .eq. | равняется        |
| /= or .ne. | не равняется     |

Логические операторы:

| Operator | Description |
| -------- | ----------- |
| .and.    | и           |
| .or.     | или         |
| .not.    | не          |
