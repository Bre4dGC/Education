В Фортране существует несколько способов манипулирования строками. Строки можно конкатенировать с помощью оператора **//**, как показано в следующем примере:
```fortran
c1 = 'Hej '
c2 = 'hopp!'
c = c1 // c2 ! = 'Hej hopp!'
```
В Фортране нет динамических строк, поэтому размер результирующей строки должен быть достаточно большим для конкатенированной строки.

Подстроки можно извлекать с помощью синтаксиса, аналогичного синтаксису, используемому при индексировании массивов.
```fortran
c3 = c(5:8) ! Содержит строку 'hopp'.
```

Частой задачей во многих кодах является преобразование чисел в строки и из строк. В Фортране нет явных функций для такого рода преобразований, поэтому для решения этой задачи можно использовать операторы **read** и **write** вместе со строками. Заменив номер файловой единицы символьной строковой переменной, можно считывать и записывать строку с помощью операторов **read** и **write**.

Для преобразования значения с плавающей точкой в строку можно использовать следующий код.
```fortran
character(255) :: mystring
real(8) :: myvalue
value = 42.0
write(mystring,'(G15.4)') value
! Теперь строка mystring содержит ' 5.676'
```

Для преобразования значения, содержащегося в строке, в значение с плавающей точкой используется оператор read-statement.
```fortran
character(255) :: mystring
real(8) :: myvalue
mystring = '42.0'
read(mystring,*) myvalue
! Теперь myvalue содержит 42,0
```

Более полный пример приведен в следующем листинге:
```fortran
program strings2
	
    implicit none
	
    integer :: i
    character(20) :: c
	
    c = '5'
    read(c,'(I5)') i
    write(*,*) i
	
    i = 42
    write(c,'(I5)') i
    write(*,*) c
	
    stop
	
end program strings2
```

Программа выдает следующий результат.
```
5
42
```
