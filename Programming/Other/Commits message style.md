### Types

| Name     | Using for                | Message about                      |
| -------- | ------------------------ | ---------------------------------- |
| Add      | new features, impl todos | feature or todo description        |
| Refactor | code, architecture       | description of optimization or etc |
| Fix      | bugs, issues             | bug description/number of issue    |
| Edit     | names                    | "old name -> new name"             |
| Delete   | code parts, sources      | name of source/parts of code       |

### Template
```
[<type>] <scope>: <message>

[add] user-dashboard: new weekly report chart feature
```

### Example
```
git commit -m "[implement] user-dashboard: new weekly report chart feature"
```
