Консольный файловый менеджер на C

По моей задумке слева будет лист с директориями, по ним можно перемещаться с помощью стрелок "вверх" и "вниз". 
Если выделена какая то папка то в правой колонке появится ее содержимое (Не будет работать в колонке с содержимым).

* Навигация между колонками осуществляется через стрелки "влево" и "вправо".
* При нажатии Enter на папке, то в той же колонке откроется ее содержимое, а в колонке с содержимым - содержимое его первого элемента.
* Если нажатие Enter было в колонке с содержимым, то, содержимое прошлой папки переместится в колонку со списком директорий, а в колонке с содержимым появится содержание папки на которой было осуществлено нажатие Enter.

Пример:
* В колонке Directory находятся папки Education, Programming, Images и т.д.
* Папка Education выделена, поэтому ее содержимое находится в колонке Content.
* Если я нажму стрелку "вправо", то перейду в колонку Content.
* Дальше при нажатии Enter на какой либо выделенной папке, содержимое Content перейдет в колонку Directory и станет главным, а содержимое папки на которой был нажат Enter перейдет в колонку Content.

## Команды
`cd <path>` - Сменить директорию
`theme <name>` - Сменить тему
`conf` - Открыть конфигурационный файл

---
## Сочетания клавиш
`Ctrl + C` - Копировать выделенный элемент
`Ctrl + V` - Вставить элемент в текущую директорию
`Ctrl + Shift + V` - Вставить элемент в выбранную директорию
`Ctrl + X` - Вырезать выделенный элемент

`Ctrl + N` - Создать текстовый файл ( ввод названия и расширения в командной строке )
`Ctrl + Shift + N` - Создать папку ( ввод названия в командной строке )
`Ctrl + R` - Переименовать выбранный элемент ( ввод в командной строке )
`Ctrl + D` - Удалить выбранный элемент

`Ctrl + F` - Найти элемент в текущем каталоге
`Ctrl + Shift + F` - Найти элемент в выбранной папке

`Ctrl + I` - Выделить элемент
`Ctrl + Shift + I` - Выделить группу элементов

`Ctrl + D` - Удалить выбранный элемент

## Спецификаторы элементов списка
`<disk>` - буква диска
`/` - папка
`.` - файл
