# Начальный экран
#### `Space + f + r` Открыть надавний файл
#### `Space + q + r` Перезагрузить последнюю сессию
#### `Space + p + p` Открыть проект
#### `Space + f + p` Открыть приватную конфигурацию
#### `Space + h + d + h` Открыть документацию

# Общее
#### `Space + f + s` Сохранить текущий файл
#### `Space + q + q` Закрыть Emacs
#### `Space + s + D` Удалить текущий файл
#### `Space + Space` Открыть файл в проекте

# Работа с текстом
#### `Alt + w` Копировать выделеный текст
#### `Alt + y` Вставить текст из буфера
